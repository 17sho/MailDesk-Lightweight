MailDesk v0.5.1 · Windows x64
================================

项目主页：https://github.com/17sho/MailDesk-Lightweight
版本页面：https://github.com/17sho/MailDesk-Lightweight/releases/tag/v0.5.1

系统要求
--------
- Windows 10/11 x64
- 可访问目标邮箱官方服务器的网络
- 仅管理你本人拥有或已经明确获授权的邮箱

两个下载包
----------
1. onefile：解压后直接运行 MailDesk.exe。
   文件少、携带方便，但首次启动需要释放 Qt 运行时，通常较慢。

2. onedir：保持 MailDesk 文件夹内容完整，运行 MailDesk\MailDesk.exe。
   启动更快，也更方便排查 Qt 插件问题，推荐日常使用。

首次运行
--------
应用数据会创建在软件同级：
MailDesk Data

更新下载和安装事务会创建在同级：
.maildesk-update

其中凭据字段使用 Windows DPAPI + Fernet 加密。邮件正文和附件不是全盘加密，
建议开启 BitLocker，并保护好 Windows 账号。

在线升级
--------
v0.3.0 是首个内置在线升级客户端的版本。v0.2.0 及更早版本需要先手动下载
并安装本版本；从 v0.3.0 开始，应用可在后台检查并下载后续正式版本。

v0.3.1 移除了安装助手对 Get-FileHash 的依赖，并强化了 PowerShell 7 与
Windows PowerShell 5 混合环境下的安装、健康回执和失败回滚兼容性。

v0.3.2 将最终文件完整性校验和安装助手接管移到后台线程，修复点击“重启并安装”
后界面看似无响应的问题；同时限制同一数据目录只运行一个 MailDesk 实例，避免多个
进程共同占用安装目录导致更新失败。

v0.4.0 及更早版本无法自动信任 v0.4.1 轮换后的发布签名，请彻底退出 MailDesk 后
人工安装本版本一次。文件夹版只替换 MailDesk 程序文件夹，必须保留同级
MailDesk Data。v0.4.1 已修复工作目录继承问题，并在失败后清理无用暂存副本；
安装本版本后，后续正式版可继续使用应用内在线升级。

v0.3.3 将 macOS arm64/x64 与 Windows 放入同一正式版本和同一签名 Release，并为
macOS 增加后台下载、确认重启、MailDesk.app 原子替换、启动健康检查与失败回滚。

v0.3.4 增加单个 HTTP/SOCKS5 代理的可视化添加与默认代理设置，并在系统设置中加入
手动检查更新入口。IMAP 收件改为每批最多下载 25 封，降低代理和海外网络下的等待时间；
遇到不兼容批量 FETCH 的服务端会自动逐封回退。

v0.3.5 增加可记忆的关闭窗口选择，可最小化到托盘或完全退出，也可在设置中恢复每次
询问。Windows 包移除未使用的 Qt 3D、PDF、Multimedia、虚拟键盘及 QML 调试资源，
同时保留 HTML 邮件、内嵌图片和远程图片所需的 QtWebEngine。

v0.3.6 修复确认“重启并安装”后停在原界面的问题。确认后不再重复联网复核 Release，
而是立即校验已经通过发布签名与 SHA-256 验证的本地文件并启动安装助手，同时记录
清晰的接管阶段日志。旧版本若仍无法重启，请手动覆盖安装 v0.3.6 一次。

v0.3.7 在邮件列表和阅读器中分别显示发件人名称与邮箱；系统设置新增全局字体、字号
和字重，并完善深色菜单、提示框及文件选择器。IMAP 连接与命令使用分级超时，下载
正文发生瞬时超时时会安全重连一次，并明确提示具体超时阶段。

v0.3.8 修复 macOS 安装包中 QtWebEngine Helper 与 Resources 目录错误导致打开邮件
阅读器时直接退出的问题，并将实际初始化邮件阅读器加入 macOS 发布前检查。

v0.3.9 将账号、邮件、设置、密钥和日志改为保存在程序同级 MailDesk Data；更新包
固定保存在同盘 .maildesk-update，更新程序目录时不会碰用户数据。更新安装使用独立
线程，真实测试覆盖 onedir 替换、重启、健康回执和数据保留。IMAP、POP3、Graph
保存传输 ID 后只收新邮件，远程图片并发自动加载；macOS 包移除未使用 Qt Framework。

v0.4.0 将关闭窗口选择改为紧凑、自适应卡片，修复大字号下的裁切、过度放大和工具栏
文字发虚。工作台图表改为轻量原生绘制，并从双平台包中移除 QtCharts、未使用的
HTTP/2/异步后端和可选压缩解码器；HTML 邮件、图片、代理和加密功能保持不变。

v0.4.1 修复“立即取件”固定只取 1 封的问题，现在单账号、批量和定时取件统一使用设置
中的每账号数量上限；同时修复 Windows 文件夹版更新目录被自身占用，并轮换离线发布
签名。便携数据迁移现在会先释放旧目录锁再清理，不再因自身锁文件阻塞启动。
v0.4.0 及更早版本需人工安装本版本一次，之后可继续正常在线升级。

v0.4.2 彻底删除独立的网络图片下载 Worker、内存缓存、加载按钮、状态条和
自动加载设置。阅读器直接渲染取件解析得到的 HTML、CID/附件图片和网络图片地址；
不再需要点击“加载网络图片”。脚本、iframe、表单、弹窗和非图片外部资源仍会被拦截。

v0.4.5 重新设计工作台“账号健康度”卡片。宽卡片采用左侧圆环与右侧状态面板，
窄窗口自动切换为上下布局；零账号状态增加完整引导文案，状态颜色、数量和占比更加
清晰，同时适配浅色、深色与字体设置。
v0.4.6 优化按需加载邮件正文的速度。账号列表同步成功后会短时保留已经认证的连接，
第一次点击正文无需再次代理握手、TLS 和邮箱登录；同一账号后续点击继续复用连接。
连接异常、配置或代理变化、账号删除以及程序退出时会自动释放旧连接。
v0.4.7 修复懒加载邮件正文后内容筛选范围不清晰的问题。筛选窗口会显示本地正文
覆盖数量，并支持“联网深度筛选”：IMAP 和 Microsoft Graph 先由服务器检索正文，
只下载命中邮件；POP3 在用户确认后逐封扫描。仅 HTML 的旧邮件也可按可见文字匹配。

v0.4.8 完成工作台、账号邮件、设置、弹窗、菜单、复选框和主题选择器的统一视觉优化；
修复大字号与窄窗口下顶部工具栏裁切、字体发虚、控件文字错位以及页面切换闪烁。
新增七套主题、恢复默认设置、可修改关闭与托盘行为，并统一流畅且可中断的轻量动画。
邮件阅读器无需 Chromium 即可异步加载公开 HTTPS 图片，图片仅驻留内存，并跳过明显
的 1×1/2×2 追踪像素；CID、data URI、附件图片及原有收发、搜索和在线升级功能保留。

v0.5.1 将“工具”菜单按邮件与帮助、维护、审计重新分组，并把邮件翻译、目标语言和
翻译前确认集中到同一入口。翻译与语言图标会跟随明暗主题同步，菜单增加清晰的键盘
焦点反馈，并保持适合中文和大字号的点击行高。本版本是由 v0.5.0 Beta 验证后推出的
正式更新；Windows 与 macOS 使用同一版本号和在线升级通道。

v0.5.0 Beta 将完整使用说明移到“工具”菜单，并使用当前 MailDesk 界面的真实截图与
编号标记说明各项操作。修复主题在明暗模式往返切换后恢复默认的问题；主题、字体和
关闭行为会跨重启及在线更新保留。主窗口、系统设置、导入、写信、阅读器、更新等
二级窗口均会按当前 Windows/macOS 屏幕自适应，并继续允许用户拖动调整大小。后台
更新完成后的确认窗口会重新相对 MailDesk 主窗口居中显示。Outlook OAuth 导入现在
也会识别同一条记录中的密码；所有密码和应用专用密码继续加密保存、遮罩显示，并且
仅在用户明确单击或选择中文右键菜单时复制到系统剪贴板。

v0.4.4 修复从模态设置窗口检查更新时，结果弹窗被遮挡而表现为“点击无反应”的问题。
设置页会先正常应用并关闭，再在主窗口前台显示更新结果。其余功能保持不变。

v0.4.3 改为邮件列表与正文两阶段同步。IMAP 和 Microsoft Graph 首次只快速获取
邮件头并立即展示列表，点击指定邮件后才下载该封正文、CID 图片和附件；POP3 优先
使用 TOP 获取邮件头，不支持 TOP 的服务器才回退完整取件。取件数量默认设为 0，
表示不限制邮件列表数量；已下载的正文会继续保留，不会被后续邮件头刷新覆盖。

发现新版时，应用会弹出提示并在顶部显示“更新”按钮。下载不会阻塞正常操作，
客户端会先用内置 Ed25519 公钥验证官方签名清单，再核对版本、安装模式、文件名、
体积和 SHA-256。只有明确确认后才会复核本地逐文件摘要并启动外部更新助手，不再
依赖第二次 GitHub 网络请求。助手完成相邻路径预复制、完整性复核和原子切换，并等待新版启动健康
回执；启动失败会恢复备份并重新打开旧版。也可以跳过当前提示版本。

更新仅从项目官方 GitHub Release 的 HTTPS 地址下载，暂存于软件同级：
.maildesk-update

在线升级需要能够访问 GitHub，且当前 Windows 用户对软件所在目录具有写入权限。
源码开发模式不会自动覆盖源码目录。Ed25519 是 MailDesk 应用级发布签名，不等同于
Windows Authenticode 证书，因此 SmartScreen 仍可能显示“未知发布者”。

安全校验
--------
下载后可在 PowerShell 中校验：

Get-FileHash .\MailDesk-v0.5.1-windows-x64-onefile.zip -Algorithm SHA256
Get-FileHash .\MailDesk-v0.5.1-windows-x64-onedir.zip -Algorithm SHA256

将结果与 Release 中的 SHA256SUMS.txt 比对。

自动更新还要求 Release 同时包含：
MailDesk-update-manifest-v1.json
MailDesk-update-manifest-v1.sig

本版本未使用商业代码签名证书。Windows SmartScreen 或杀毒软件可能对新的
PyInstaller 程序显示未知发布者提示；请先核对 SHA256，并仅从项目官方 Release 下载。
