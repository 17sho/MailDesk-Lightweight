# Changelog

## [0.5.0] - 2026-07-18 (Beta)

### Added

- 在“工具”菜单中加入可独立打开的完整使用说明，并使用真实 MailDesk 截图和编号标记。
- 账号表新增“密码/授权码”遮罩列；用户可单击或通过中文右键菜单复制完整凭据。
- Outlook OAuth 批量导入支持同时识别并保留密码、Client ID 和 Refresh Token。

### Changed

- 主窗口及导入、写信、阅读器、设置、更新等二级窗口会按当前屏幕自适应，并支持自由调整大小。
- 后台更新完成后的确认窗口始终相对 MailDesk 主窗口居中显示。

### Fixed

- 修复明暗模式往返切换后丢失已选主题的问题；主题和字体设置会跨重启、跨更新保留。
- 修复大字号或较小屏幕下设置底栏和二级窗口内容被裁切的问题。
- 修复 Outlook 自由文本导入中密码可能被误判为 OAuth 字段的问题。

### Security

- 密码、应用专用密码和 OAuth 令牌继续使用设备保护密钥加密存储；界面和日志不会显示完整凭据。
