# MailDesk v0.3.2

这是在线升级可靠性修复版本。

## 本版修复

- 修复点击“重启并安装”后，界面在校验数千个 onedir 文件期间看似没有反应的问题。
- 最终发布状态复核、暂存文件完整性校验和安装助手接管均在后台执行，主界面保持响应。
- 更新窗口现在明确显示“复核发布状态”“校验安装文件”和“安装助手已接管”等阶段。
- 确认弹窗显示期间不再提前把按钮改成“正在启动安装”，取消后可正常再次操作。
- 增加单实例锁，阻止多个 MailDesk 进程同时访问本地数据库或占用同一安装目录。
- 保留 Ed25519 发布签名、SHA-256、事务锁、启动健康回执和失败回滚机制。

## 下载建议

- 推荐日常使用 `MailDesk-v0.3.2-windows-x64-onedir.zip`，启动速度更快。
- 需要单个可执行文件时使用 `MailDesk-v0.3.2-windows-x64-onefile.zip`。
- 下载后可使用 Release 中的 `SHA256SUMS.txt` 核对文件。

## 升级提示

如果电脑上已经运行了多个旧版 MailDesk，请先通过各窗口托盘菜单完全退出，只保留一个实例后再执行在线升级。v0.3.2 开始会自动阻止重复实例。
