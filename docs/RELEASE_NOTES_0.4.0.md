# MailDesk v0.4.0

这是一次以界面清晰度、响应式布局和双平台包体精简为重点的正式更新。

## 界面与字体

- 关闭窗口选择改为紧凑的双卡片布局，移除固定大高度和无效空白。
- 选项标题、说明、图标和焦点状态分层呈现，支持键盘操作与托盘不可用提示。
- 字号调整由成比例放大改为层级偏移，18pt 下标题不再被放大到 45px。
- 工具栏、菜单和按钮使用 Qt 稳定字重，移除 650、720、750、800 等跨平台映射不一致的值。
- 用户可继续设置 9–18pt 字号与标准、清晰、半粗三档正文字重。

## 简化与包体

- 工作台账号健康度与收件趋势改用原生 `QPainter` 绘制，移除 QtCharts 及其 OpenGLWidgets 运行库。
- Windows 只保留实际使用的平台、图片、TLS、图标和网络状态 Qt 插件。
- WebEngine 语言包只保留简体中文与英文；完整 HTML 邮件、内嵌/远程图片和附件阅读能力不变。
- 移除 HTTPX 未使用的 HTTP/2、Trio、WebSocket、Brotli 和 Zstandard 可选运行库；Graph、OAuth、Webhook 与更新继续使用标准 HTTP/1.1 gzip/deflate。
- SOCKS5/HTTP 代理仍由小型路由与 socket 适配层实现，没有内置代理服务器。

## 验证

- 344 项自动化测试通过，3 项仅在对应原生平台执行。
- Windows onedir 打包后的 WebEngine、HTTPX 与 SOCKS 运行时启动探针通过。
- 以 v0.3.9 onedir 为基线，解包目录由约 333.9 MB 降至约 324.0 MB，测试 ZIP 由约 139.8 MB 降至约 134.3 MB。正式发行包还会包含许可证和说明文件，最终大小以 Release 资产为准。
