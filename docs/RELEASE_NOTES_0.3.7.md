# MailDesk v0.3.7

本版本改进邮件发件人展示、全局字体可读性、深色模式一致性，并增强 IMAP 瞬时超时恢复能力。

## 主要变化

- 邮件数据库新增发件人名称字段；IMAP 与 Microsoft Graph 会同时保存发件人名称和邮箱地址。
- 主邮件列表、工作台最近邮件和独立阅读器明确展示发件人名称与邮箱；旧邮件缺少名称时仍会显示邮箱。
- 系统设置新增“外观与字体”，支持系统推荐字体、9–18pt 字号，以及标准、清晰、半粗、粗体四档字重。
- 字号设置会同步缩放主题中原先固定的小号文字，并即时应用到主窗口、设置、菜单和工具区域。
- 深色模式增加统一 Qt 调色板，并完善菜单、工具提示、消息框、文件选择器、进度条和横向滚动条的颜色与对比度。
- IMAP 连接超时保持 20 秒，登录、目录和正文命令超时调整为 90 秒，避免把较大邮件下载误判为连接失败。
- 无副作用的 IMAP 取件遇到瞬时超时时会关闭已损坏连接并安全重连一次；日志和界面会指出连接、登录、文件夹、搜索或正文下载阶段。
- 数据库平滑迁移到 schema v7，不会清空已有账号、邮件、附件或设置。

## 发布平台

- Windows x64：onefile 与 onedir ZIP。
- macOS 13+：Apple Silicon arm64 与 Intel x64 的 DMG，以及应用内更新 ZIP。
- 四种安装模式使用同一 v0.3.7 正式 Release 和同一 Ed25519 签名更新清单。
