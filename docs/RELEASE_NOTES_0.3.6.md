# MailDesk v0.3.6

本版本修复确认“重启并安装”后长时间没有反应的问题。

## 主要变化

- 移除确认安装后的重复 GitHub Release 联网复核，避免网络慢、GitHub API 限流或连接异常阻止安装助手启动。
- 下载阶段仍完整执行 Ed25519 发布签名、GitHub SHA-256、文件大小和安全解压校验，安全等级不降低。
- 确认后立即切换为“正在准备安装”，在后台复核本地暂存文件并启动外部安装助手。
- 更新窗口实时显示“校验暂存文件”和“启动外部安装助手”两个阶段。
- 更新接管、失败原因和阶段状态写入 `logs/app.log`，便于排查。
- 保留安装助手握手、原子替换、启动健康检查和失败回滚。

## 旧版本说明

如果 v0.3.5 及更早版本点击“重启并安装”仍停在原界面，请从 Release 页面手动下载并覆盖安装 v0.3.6 一次。安装 v0.3.6 后，后续版本可以使用修复后的在线升级流程。
