# MailDesk v0.4.1

这是 v0.4 系列的稳定性收尾正式版，重点修复取件数量和 Windows 文件夹版更新流程，
并重新验证 Windows/macOS 双平台构建。

## 取件逻辑

- “立即取件”只限定当前账号，不再把邮件数量强制改成 1。
- 设置中的“每账号每次最多新增”统一用于立即取件、批量取件和定时取件。
- 设置为 0 表示不限制；增量取件仍会跳过本地已保存邮件，避免重复下载。
- IMAP 每批最多获取 25 封，Graph 每页最多 50 封并按需翻页，POP3 同样遵循设置上限。

## Windows 更新修复

- 安装助手不再继承待替换的 `MailDesk` 工作目录，修复 onedir 目录被自身占用而回滚。
- 目录切换支持短时重试，失败后清理不再需要的暂存副本。
- 安装后继续等待新版健康回执；启动失败仍会自动恢复旧版并保留用户数据。
- 便携数据迁移会先释放旧目录锁再清理副本，避免 Windows 被自身锁文件阻塞。
- 启动探针和开发测试指定的独立数据目录不再导入用户的真实旧版数据。

## 发布签名轮换

- 原 v0.4.0 离线私钥不可用，v0.4.1 启用新的 Ed25519 发布公钥。
- v0.4.1 同时信任旧公钥和新公钥；新私钥仅以 Windows DPAPI 加密形式离线保存。
- v0.4.0 及更早版本需要人工安装 v0.4.1 一次；从 v0.4.1 开始恢复正常在线升级。

## 验证范围

- 全套单元、服务、集成和 GUI 自动化测试。
- Windows onefile/onedir WebEngine 邮件阅读器启动探针。
- Windows 更新助手的成功替换、健康检查、失败回滚和用户数据库保留测试。
- macOS Intel/Apple Silicon 原生构建、架构检查、WebEngine 启动和更新 ZIP 校验。

真实邮箱登录仍取决于用户提供的有效授权码/OAuth 授权、服务商协议开关和网络环境。
