## MailDesk v0.4.8

这是 Windows 与 macOS 同步发布的正式版本，保留原有邮箱导入、取件、搜索、发件、附件、代理、分组、翻译和在线升级能力。

### 界面与交互

- 统一优化工作台、账号与邮件、系统设置、添加邮箱、导入、代理、发件、更新、关闭确认等界面。
- 修复复选框状态、控件文字对齐、字体发虚和字重未全局生效的问题。
- 大字号及窄窗口下，顶部工具栏会自适应隐藏次要说明，系统设置等主要操作保持可见。
- 新增七套主题、恢复默认设置和可修改的关闭/托盘行为。
- 页面、主题、弹窗及运行日志动效改为轻量、可中断的平滑动画，减少闪烁和黑块。

### 邮件阅读器

- 修复 HTML 邮件中远程 Logo 和正文图片显示为破损占位的问题。
- 不引入 Chromium、QtWebEngine 或浏览器插件，使用现有网络栈在后台加载公开 HTTPS 图片。
- 远程图片只驻留内存，不写入用户数据目录；同一邮件重绘时复用内存缓存。
- 跳过声明为 1×1/2×2 的明显追踪像素，并限制协议、重定向、数量、体积和像素规模。
- CID、data URI、普通附件图片和两阶段 IMAP 正文加载逻辑保持不变。

### 验证与升级

- Windows x64 提供 onefile 与推荐的 onedir 压缩包。
- macOS 同步提供 Apple Silicon（arm64）与 Intel（x64）的 ZIP/DMG。
- 两个平台使用同一份 Ed25519 签名更新清单；在线升级保留软件同级 `MailDesk Data`。
- 发布前全量测试结果：415 通过，3 跳过。

macOS 包当前仍未使用 Apple Developer ID 公证，首次启动请按仓库中的 macOS 说明操作。
